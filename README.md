# Veris
Veris is a local Chrome/Edge extension for:  - AI text possibility detection - AI image possibility detection.

> Privacy-first On-Device AI Chrome Extension for smarter web browsing.

## Problem Statement
Most AI browser assistants rely on cloud processing, which can compromise user privacy, increase latency, and require a constant internet connection.

## Solution
Veris is an On-Device AI Chrome Extension that performs AI inference locally in the browser, enabling fast, secure, and privacy-focused assistance without sending sensitive user data to external servers.

## Features
- 🚀 On-device AI inference
- 🔒 Privacy-first architecture
- ⚡ Fast response time
- 🌐 Chrome Extension
- 📄 Smart webpage understanding
- 🧠 AI-powered search and assistance
- 🎨 Clean and responsive UI

## AI Models & Tech Stack

### AI Models
- Gemma 3n 
- BGE Small Embedding Model 

### Runtime & Frameworks
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Chrome Extension (Manifest V3)
- Transformers.js
- ONNX Runtime Web
- WebGPU

## On-Device AI
The core AI functionality runs locally on the user's device. No sensitive prompts or webpage content are sent to external AI servers, ensuring better privacy, lower latency, and offline-friendly functionality.

## Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/veris.git
```

2. Open Chrome and go to:
```
chrome://extensions
```

3. Enable **Developer Mode**.

4. Click **Load Unpacked**.

5. Select the project folder.

## Usage

1. Open any webpage.
2. Launch the Veris extension.
3. Use the available AI features.
4. Get AI-powered results directly on your device.
